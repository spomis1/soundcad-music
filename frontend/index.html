¿Qué hace?

SoundCard agrega datos de 7 APIs en tiempo real para darte un perfil completo de cualquier artista o canción: seguidores, giras históricas, discografía, videos, Wikipedia, samples y un Momentum Score propio.

Momentum Score

Score 0–100 calculado a partir de: Spotify followers (30 pts), popularidad en Spotify (30 pts), views de YouTube top-5 (20 pts) e historial de giras vía Setlist.fm (20 pts).

Fuentes de datos
  • Spotify — followers, popularidad, discografía, top tracks
  • Last.fm — plays totales, oyentes, artistas similares
  • YouTube — top videos del canal oficial, views
  • Wikipedia — biografía, pageviews mensuales
  • MusicBrainz — país, tipo de artista, samples
  • Setlist.fm — historial completo de giras
  • Apple Music — link directo al artista
Stack técnico
  • Backend — Python + FastAPI en Render.com
  • Base de datos — PostgreSQL en Neon.tech (cache 7 días)
  • Frontend — Vanilla JS en GitHub Pages
  • Código — github.com/spomis1/soundcad-music
