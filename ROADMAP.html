V3 — Búsqueda por canción Gratis 2-3 días

Agregar un segundo modo de búsqueda: por canción. El problema es que hay canciones con el mismo nombre, featurings, re-grabaciones y covers — cada una es un objeto distinto en la industria. La solución es usar MusicBrainz como fuente de IDs únicos, completamente gratis.

🆔
ISRC + MBID: los IDs únicos de la industria ISRC (International Standard Recording Code) es el estándar global — cada grabación tiene uno. MBID (MusicBrainz Recording ID) es un UUID gratuito y abierto que identifica la misma cosa. Un featuring ("Blinding Lights ft. X") tiene su propio ISRC y MBID distinto al original.
🏛️
MusicBrainz API — sin API key, gratis total GET https://musicbrainz.org/ws/2/recording?query=Halo artist:Beyoncé&fmt=json
Devuelve lista de grabaciones con MBID, ISRC, duración, año, artistas — perfecta para desambiguar. Rate limit: 1 req/seg (respetar con asyncio.sleep(1)).
🔍
Flujo de búsqueda con desambiguación Usuario escribe "Halo" → backend llama MusicBrainz → devuelve lista de 5-10 resultados con artista + año + duración → frontend muestra el listado para que el usuario elija → se carga el MBID elegido como ID único para todo lo demás.
👥
Créditos: escritores, productores, featurings MusicBrainz tiene relaciones Work→Recording con roles: compositor, letrista, productor. GET /recording/{mbid}?inc=artist-credits+work-rels devuelve todos los créditos sin costo.
📝
Genius API — letra + anotaciones Free tier de Genius permite buscar por artista + título y obtener la URL de la letra. API key gratis en genius.com/api-clients. Complementa MusicBrainz: créditos técnicos + texto completo de la letra.
🎵
Canciones relacionadas — Last.fm track.getSimilar track.getSimilar&artist=X&track=Y&limit=10 devuelve tracks parecidos con score de similitud. Gratis, usa la misma API key de Last.fm que ya tenés.
🗄️
Nueva tabla: song_cache Misma lógica de caché de 7 días, pero indexada por MBID (UUID único). Schema: mbid, isrc, title, artist, year, credits_json, lyrics_url, similar_json, cached_at. ~20KB por canción = espacio prácticamente ilimitado en el plan Railway free.
⚠️
Samples — limitación conocida WhoSampled API es de pago (~$50/mes). MusicBrainz tiene datos parciales de samples en sus relaciones Work-Work. No es confiable al 100% pero es lo único gratuito disponible. Se documenta como limitación en el README.
